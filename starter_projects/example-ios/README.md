### Dependencies
From the command line cd into /example-ios and run:

```shell
bash get_ios_dependencies.sh
```


### Prerequisites

You will need

- IntelliJ IDEA
- AIR 33.1.1.217+
- Xcode 12.3
- wget on macOS via `brew install wget`
- [Carthage](https://github.com/Carthage/Carthage#installing-carthage)
