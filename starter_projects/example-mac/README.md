### Dependencies
From the command line cd into /example-mac and run:

```shell
bash get_mac_dependencies.sh
```

### Prerequisites

You will need

- IntelliJ IDEA
- AIR 33.1.1.217+
- Xcode 12.5
- wget on macOS via `brew install wget`
- [Carthage](https://github.com/Carthage/Carthage#installing-carthage)
